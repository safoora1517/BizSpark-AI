Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   ```

2. Navigate to the project folder:
   ```bash
   cd <repo-name>
   ```

3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:

   ```bash
   npm run dev
   ```
---
